# Banana Recipes - Fabric Mod for Minecraft Java Edition

A Fabric mod that adds bananas and banana-based recipes to Minecraft Java Edition!

## Features

This mod adds the following items to Minecraft:
- **Banana**: A basic food item that restores 4 hunger points
- **Golden Banana**: A special banana crafted with gold that provides Regeneration and Absorption effects
- **Banana Bread**: A filling food item made with bananas and wheat
- **Dried Banana**: A quick snack made by smelting or smoking bananas

## Recipes

- **Golden Banana**: Craft a banana surrounded by 8 gold ingots (similar to golden apple)
- **Banana Bread**: Craft with 3 bananas on top and 3 wheat on bottom
- **Dried Banana**: Smelt or smoke a banana in a furnace or smoker

## Building

### Prerequisites
- Java 21 or higher
- Internet connection (to download Minecraft and Fabric dependencies)

### Build Instructions

1. Clone this repository
2. Run `./gradlew build` (Unix/Mac) or `gradlew.bat build` (Windows)
3. The compiled mod JAR will be in `build/libs/`

## Installation

1. Install [Fabric Loader](https://fabricmc.net/use/) for Minecraft 1.21
2. Download [Fabric API](https://modrinth.com/mod/fabric-api) for Minecraft 1.21
3. Place both Fabric API and this mod's JAR file in your `.minecraft/mods` folder
4. Launch Minecraft with the Fabric profile
