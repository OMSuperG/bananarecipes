# Textures

This mod requires the following 16x16 PNG textures to be placed in `src/main/resources/assets/bananarecipes/textures/item/`:

1. **banana.png** - A yellow banana sprite
2. **golden_banana.png** - A golden/shiny banana sprite (similar styling to golden apple)
3. **banana_bread.png** - A loaf of bread with banana coloring
4. **dried_banana.png** - A darker, dried-looking banana sprite

## Icon

The mod also needs an icon file at `src/main/resources/assets/bananarecipes/icon.png` (512x512 PNG recommended).

These textures should follow Minecraft's art style and be 16x16 pixels for items.
